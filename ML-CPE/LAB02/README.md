Summary of the Project

LAB1: Dataset Exploration
Loaded the CSV dataset into Python.
Checked the dataset shape (rows and columns).
Checked data types of each column.
Displayed summary statistics.
Checked for missing values.
Checked for duplicate records.
Displayed the class distribution.

LAB2: Data Visualization
Created a Histogram to visualize the distribution of the data.
Created a Correlation Heatmap to analyze the relationship between numerical features.

Part 3: Data Cleaning
Filled missing values using the mean.
Removed duplicate records.
Corrected incorrect data (e.g., changed "YES" to "Yes").
Converted columns to the correct data types.
Compared the Mean and Median of the numerical data.

Part 4: Feature Engineering
Applied Label Encoding to convert text labels into numbers (e.g., No = 0, Yes = 1).
Applied One-Hot Encoding to convert categorical data into binary (0/1) columns.
Overall Conclusion

This project demonstrates the basic steps of Data Preprocessing before building a Machine Learning model. The data was explored, cleaned, transformed, and encoded to improve its quality and prepare it for model training.
